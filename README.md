# 🚀 Streamlit SaaS App with Stripe Subscription

A complete **SaaS starter template** built using Streamlit.

## ✨ Features

* 🔐 Authentication (Login/Register)
* 💳 Stripe Subscription Payments
* 📊 Dashboard UI
* 👤 User session management
* 🗄️ SQLite database
* 💎 Premium feature locking

## ⚙️ Installation

```bash
pip install streamlit stripe sqlalchemy passlib pandas
```

## ▶️ Run

```bash
streamlit run app.py
```

## 🔑 Setup Stripe

Replace:

```python
STRIPE_SECRET_KEY = "your_key"
STRIPE_PRICE_ID = "your_price"
```

## 📁 Structure

Single file app:

* app.py
